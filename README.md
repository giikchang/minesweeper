# 지뢰찾기 (Minesweeper) — Python (Windows/Tkinter)

간단한 윈도우용 지뢰찾기 게임입니다. Tkinter 기반의 단일 파이썬 파일로 동작하며, 초급/중급/고급 난이도를 지원합니다.

## 주요 기능
- 초급(9×9, 10지뢰), 중급(16×16, 40지뢰), 고급(16×30, 99지뢰) 난이도
- 첫 클릭 보호(첫 클릭 주변에는 지뢰가 배치되지 않음)
- 왼쪽 클릭: 칸 열기 (0은 자동 확장)
- 우클릭: 깃발 토글(남은 깃발 수 표시)
- 타이머, 재시작 버튼, 상태 표시 및 승리/패배 팝업
- PyInstaller로 exe 빌드 가능

## 요구사항
- Python 3.6 이상
- Tkinter (Windows의 표준 Python 설치에는 포함되어 있음)
- (선택) PyInstaller — 실행 파일로 빌드하려면 설치

## 설치 및 실행
1. 저장소를 클론하거나 파일을 다운로드합니다.
2. Python이 설치되어 있는지 확인합니다.
3. 다음 명령으로 실행합니다:

```bash
python minesweeper.py
```

Windows에서 double-click으로 실행하려면 `.py` 파일과 Python 파일 연결이 되어 있어야 하거나, 아래와 같이 exe로 빌드하세요.

## exe로 빌드 (선택)
PyInstaller를 사용해 단일 실행 파일을 만들 수 있습니다.

```bash
pip install pyinstaller
pyinstaller --noconsole --onefile minesweeper.py
```

생성된 exe는 `dist/minesweeper.exe` 에 위치합니다.

## 사용 방법 (간단)
- 왼쪽 클릭: 칸 열기. 숫자는 주변 지뢰 수.
- 우클릭: 깃발 표시 / 해제.
- 메뉴에서 난이도 변경 또는 재시작 가능.
- 모든 비폭탄 칸을 열면 승리, 폭탄을 누르면 패배.

## 구성 및 코드 위치
주요 코드 파일:
- `minesweeper.py` — 게임 전체 구현(Tkinter UI, 게임 로직 등)

필요한 기능(예: 아이콘, 로컬 점수 기록, 사운드 등)은 `minesweeper.py`를 수정해 확장할 수 있습니다.
